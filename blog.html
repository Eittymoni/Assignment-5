1. What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, allowing developers to manipulate the content, structure, and style of a webpage dynamically using JavaScript.

2. How do you select an element from the DOM?

document.getElementById("id"), document.querySelector("selector"), document.querySelectorAll("selector"), document.getElementsByTagName("tagName"), document.getElementsByClassName("className")

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element to manage events for its child elements. It is useful because: Reduces the number of event listeners, Automatically handles newly added child elements.

4. How do you manipulate an element's attributes and styles using the DOM?

element.setAttribute("attr", "value"), element.getAttribute("attr"), element.removeAttribute("attr"), element.style.property = "value" or element.classList.add/remove/toggle("class")
